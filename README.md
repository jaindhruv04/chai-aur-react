# Chai Aur React Projects

This repository contains multiple React projects built while learning React through the Chai Aur React series.

Each project is kept in a separate folder and can be run independently.

| 06 | Currency Convertor | A currency converter app using custom hooks and currency data | [View Project](https://jaindhruv04.github.io/chai-aur-react/06currencyConvertor/) |
